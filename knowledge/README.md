# Immigration Petition Knowledge Base

Reference playbook for O-1A, EB-1A, and EB-2 NIW petitions. O-1A and EB-1A share the same criteria, so they share the same knowledge files. Gets smarter with every case.

## Structure

```
knowledge/
└── shared/
    ├── overview-o1a-eb1a.md     # Standard, Kazarian framework, adjudicator mindset
    ├── criteria/
    │   ├── 01-awards.md         # Each criterion: legal standard, evidence hierarchy,
    │   ├── 02-membership.md       argument patterns, RFE lessons (real USCIS language),
    │   ├── 03-published-material.md   things to watch for
    │   ├── 04-original-contributions.md
    │   ├── 05-scholarly-articles.md
    │   ├── 06-critical-employment.md
    │   ├── 07-high-remuneration.md
    │   └── 08-judging.md
    ├── evidence-hierarchy.md    # Universal evidence weighting (Tier 1-4)
    ├── argument-patterns.md     # Reusable argument structures
    └── uscis-policy-alerts.md   # Key policy citations (PA-2022-03, Kazarian, Dhanasar)
```

## How to use

1. **Before drafting** — read the relevant criterion files. Check evidence hierarchy, RFE lessons, and things to watch for.
2. **After an RFE** — add the real USCIS language to the relevant criterion file under "RFE Lessons." Quote exactly.
3. **After an approval** — add anonymized winning argument patterns.

## Rules

- **No client PII** — anonymize all case references
- **RFE lessons must quote real USCIS language** — not our interpretation of what USCIS might say
- **Update, don't duplicate** — if a pattern already exists, refine it
